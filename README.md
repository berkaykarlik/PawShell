# PawShell / pawsh

Dog themed shell. Because world needs one.

## Capabilities

* executes entered command, prints the output and waits for the next command,
* until user use `exit` command to close it.

## Requirements
for Ubuntu:
```shell
sudo apt update
sudo apt install g++ make
```

## Build

```shell
make
```

## Usage

```shell
./pawsh
```

## Clean

```shell
make clean
```
